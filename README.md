# REST API for E-Commerce Application using Typescript

## Setup

1. ```bash
    npm init -y
   ```

1. ```bash
    npm i express
   ```

1. ```bash
    npm i -D ts-node-dev
   ```

1. ```bash
    npm i dotenv

   ```

1. ```bash
    npm i -D jest typescript @types/jest ts-jest

   ```

1. ```bash
    npx ts-jest config:init

   ```
